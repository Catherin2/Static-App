Assignment 3 : Group 5 API

Find Group 5 API endpoints
1. User Management Service User Registration POST /api/auth/register
Registers a new user. 2. User Authentication POST /api/auth/login Authenticates a user and returns a JWT token.
Resource Catalog Service 3. Retrieve Extension List GET /api/extensions Returns a list of available extensions with filtering options.
4. Retrieve Extension Details GET /api/extensions/{id} Returns detailed information about a specific extension.
5. Add a New Extension POST /api/extensions Allows authenticated users to add a new extension.
6. Update Extension Information PUT /api/extensions/{id} Updates extension details, including archiving.
7. Delete an Extension DELETE /api/extensions/{id} Removes an extension from the catalog.
File Upload Service 8. Upload an Extension File POST /api/file/upload Uploads an extension file to the server.
9.Download an Extension File GET /api/file/download/{id} Downloads an extension file and tracks download count.
